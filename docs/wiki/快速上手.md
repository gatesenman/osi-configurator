# 快速上手

## 方式一：桌面版（推荐）

到 [Releases](https://github.com/gatesenman/osi-configurator/releases) 下载对应平台安装包：

| 平台 | 文件 | 说明 |
| --- | --- | --- |
| Windows | `OSI.Configurator.Setup.x.x.x.exe` | NSIS 安装器 |
| macOS | `OSI.Configurator-x.x.x-universal.dmg` | Intel / Apple Silicon 通用 |
| Linux | `OSI.Configurator-x.x.x.AppImage` | 免安装，`chmod +x` 后直接运行 |

> 安装包未做代码签名：Windows 首次运行需在 SmartScreen 中点「仍要运行」；macOS 首次打开需右键 →「打开」。

## 方式二：本地开发

环境要求：Node.js >= 20，pnpm 10。

```bash
git clone https://github.com/gatesenman/osi-configurator.git
cd osi-configurator
pnpm install
pnpm dev        # http://localhost:3000
```

## 第一个语义模型

1. **模型分区**：填写模型 name（snake_case）、description，按需开启 ai_context
2. **数据集分区**：「添加数据集」→ 填 name 与 source（物理表全名 `db.schema.table`）→ 添加字段，维度字段选择维度形态（普通 / 时间）
3. **关系分区**：定义数据集间关联（from 多端 → to 一端 + 列映射）
4. **指标分区**：「添加指标」→ 填聚合表达式（至少一条 ANSI_SQL 方言）
5. 右侧实时生成规范文件，底部显示 **官方 Schema 校验** 与 **语义 Lint** 状态
6. 通过右上角复制 / 下载按钮导出 `<模型名>.osi.yaml`

## 数据安全

- 所有编辑自动保存在本机浏览器（localStorage），刷新不丢失
- 「版本」按钮可命名保存快照，随时恢复或对比
- 不上传任何数据到服务器；AI 功能仅在你主动配置提供商并点击生成时调用你指定的 API
